# wazuh_rules_for_freeipa

This repository contains Wazuh correlation rules for detecting next attacks on FreeIPA:
* Using suspicious attributes in request to LDAP
* Shell attribute modification
* Ssh key attribute modification
* HBAC rule creation or modification
* Domain data export by Directory Manager account
* Sudo rule creation or modification
* Password spray attack via Kerberos
* Brute force attack via Kerberos
* Password spray attack via LDAP
* Brute force attack via LDAP
* KeyRing access
* Keytab files access
